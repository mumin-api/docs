# Changelog

All notable changes to Mumin Hadith API will be documented here.

## [2.5.0] - 2026-03-05

### Added

- 🤖 **MuminAI Scholarly Engine** - Highly accurate hadith explanations and context.
- 📚 **Enriched Metadata** - Hadith responses now include secondary scholarly insights in the `metadata` field.
- 🌍 **Expanded Language Intelligence** - Improved AI-driven support for Russian, Uzbek, Turkish, and Urdu.

### Changed

- Updated Statistics: Now serving **40k+** authentic hadiths.
- Enhanced Collection Metadata with professional scholarly descriptions.

## \[2.0.0\] - 2024-01-10

### Added

- 🎉 **Multi-language support** - Arabic, English, Russian, Uzbek
- 🔍 **Full-text search** - Search across all hadiths
- 📊 **Analytics dashboard** - Track your API usage
- 🔐 **Enhanced security** - Fraud detection and IP whitelisting
- 💳 **Flexible billing** - Pay-as-you-go pricing

### Changed

- Improved response times (now &lt;50ms with Redis caching)
- Updated API key format to `sk_mumin_...`
- Enhanced error messages with actionable codes

### Fixed

- Search accuracy for Arabic text
- Rate limiting edge cases
- Pagination offset calculation

## \[1.0.0\] - 2023-12-01

### Added

- Initial release
- Basic hadith endpoints
- API key authentication
- Rate limiting
