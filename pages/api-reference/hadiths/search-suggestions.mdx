# Search Suggestions

Provides topic-based autocomplete suggestions for hadith searches.

<ApiEndpoint method="GET" path="/hadiths/suggestions" auth="required" />

## Query Parameters

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `q` | string | Yes | The partial search query |
| `language` | string | No | The language code (default: `en`) |

## Response

Returns an array of suggested search concepts.

```json
[
  "prayer",
  "prayer times",
  "prayer in congregation"
]
```
