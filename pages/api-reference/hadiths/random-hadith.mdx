# Random Hadith

Get a random hadith from the library. You can optionally filter by collection or grade.

<ApiEndpoint method="GET" path="/hadiths/random" auth="required" />

## Query Parameters

| Parameter    | Type    | Default | Description                                   |
| ------------ | ------- | ------- | --------------------------------------------- |
| `lang`       | string  | `"en"`  | Language code                                 |
| `collection` | string  | -       | Filter by collection slug (e.g., `sahih-bukhari`) |
| `grade`      | string  | -       | Filter by grade (e.g., `Sahih`, `Da'if`)      |
