### Create an Account

### Get Your API Key

1. Navigate to **API Keys** in your dashboard
2. Click **"Create New Key"**
3. Copy your API key (starts with `sk_mumin_`)

<Callout type="warning">
  **Important:** Store your API key securely. It will only be shown once!
</Callout>

  <Tab label="Python">
    ```python
    import requests

    headers = {'Authorization': 'Bearer YOUR_API_KEY'}
    response = requests.get(
        'https://api.mumin.ink/v1/hadiths/1',
        params={'lang': 'en'},
        headers=headers
    )
    hadith = response.json()
    print(hadith['translations']['en']['text'])
    ```
  </Tab>
  
  <Tab label="PHP">
    ```php
    <?php
    $ch = curl_init();
    curl_setopt($ch, CURLOPT_URL, 'https://api.mumin.ink/v1/hadiths/1?lang=en');
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        'Authorization: Bearer YOUR_API_KEY'
    ]);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    
    $response = curl_exec($ch);
    $hadith = json_decode($response);
    
    echo $hadith->translations->en->text;
    ?>
    ```
  </Tab>
  
  <Tab label="cURL">
    ```bash
    curl -X GET "https://api.mumin.ink/v1/hadiths/1?lang=en" \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Tab>
